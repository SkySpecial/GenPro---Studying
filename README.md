###Окружение "picture" и работа с некоторыми графическими объектами в tex'е:

\begin{picture} (Ширина, Высота) - параметры графического бокса, причём сам бокс - очередная буква, символ новой строки. Поэтому где окружение разместим в исходном файле, там он и появится и из-за своих параметров раздвинет строки вокруг.

(x0, y0) - задаём к-ты левого нижнего угла. Сам угол не гуляет(!) по странице из-за смены к-т, он завязан с расположением всего бокса по предыдущему пункту. Относительно этих к-т система определяет важную точку (0, 0). Если этот необязательный параметр опускаем, значит, левый нижний угол и есть (0, 0).

Текст в окружении вследствие неимения параметров размещается в текущей нулевой точке. Вообще любой объект без команды \put ставится автоматически в (x0, y0). При этом, если есть несколько таких сущностей, то они по очереди сдвигаются по горизонтали, аналогично с вертикальным размещением отметок на полях.
